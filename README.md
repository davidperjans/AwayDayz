# ⚽ AwayDayz – Webbapplikation för att logga arenabesök

## 📖 Beskrivning
**AwayDayz** är en webbapplikation för fotbollsfans som vill logga sina besök på olika arenor. Användare kan registrera sig, spara sina besök, ladda upp bilder och skriva kommentarer om sina upplevelser. Systemet lagrar all data i en **SQL Server (SSMS)**-databas och möjliggör en personlig upplevelse genom statistik och historik över besökta arenor.

## 🚀 Funktioner
- 🏟️ **Arenabesök** – Lägg till och hantera dina besökta arenor.
- ⭐ **Betyg & Recensioner** – Betygsätt arenor och skriv kommentarer.
- 📸 **Bilduppladdning** – Ladda upp och visa bilder från dina besök.
- 👤 **Användarkonto** – Skapa en profil och hantera dina uppgifter.
- 📊 **Statistik & Historik** – Se tidigare besök och favoritarenor.
- 🔍 **Sök & Filtrering** – Hitta specifika arenor och besök.

## 🛠️ Teknologi
- **Frontend:** Blazor eller .NET MAUI
- **Backend:** C# ASP.NET Core Web API
- **Databas:** SQL Server (SSMS) eller PostgreSQL
- **Autentisering:** ASP.NET Identity eller OAuth 2.0

## 📅 Projektstruktur & Arbetsprocess
Projektet följer en **agil utvecklingsprocess** med iterativa sprintar, backlog och kontinuerlig utveckling. Funktioner prioriteras utifrån feedback och förbättringsförslag.
